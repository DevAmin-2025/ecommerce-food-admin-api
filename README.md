This is the Admin Panel Api for an eCommerce food ordering platform built with Laravel 12. It provides backend management features including CRUD operations for products, categories, coupons, orders, content pages, and more. The panel is built to work alongside a customer-facing website.

## 🔑 Authentication bearer
Note that all End-points except **Login** need **bearer token**. You'll be given a token once you logged in.
